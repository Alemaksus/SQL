Курсовая работа Максакова Александра по курсу "Основы РБД. MySQL."

Проект РБД povar (далее - БД) онлайн-сервиса для подбора и обмена рецептами разных стран для составления меню и списка продуктов.

Основная цель: ввод, хранение, поиск рецептов разных стран мира, составление из них меню, списков покупок
и общение между посетителями сайта.

Задачи, решаемые базой данных:

1. Составление профайла пользователем;
2. Создание им новых рецептов, добавление ингридиентов, фотографий;
3. Общение между пользователями через сообщения;
4. Подбор рецепта по стране, автору, времени приготовления, пищевой ценности;
5. Возможность лайкать понравившиеся рецепты;
6. Возможность подбора личного меню с любимыми рецептами в виде отдельной таблицы menu;
8. Создание списка ингридиентов из таблицы menu и количества порций. 

Структура курсовой работы: 

1. Краткое описание курсовой работы: 1_project_description.txt;
2. Скрипт создания БД (14 таблиц): db_povar.sql;
3. ERDiagram: ERDiagram_povar.mwb; 
4. Скрипт наполнения БД: data_insert.sql
5. Примеры характерных выборок: selects_from_povar.sql;
6. Представления (3шт): views.sql;
7. Хранимые процедуры (2шт): procedures_povar.sql;
8. Триггеры (10шт): triggers.sql.
